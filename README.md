        </p>

        <h2>Key Features</h2>
        <div class="features">
            <ul>
                <li>Comprehensive system tailored for team contact management needs.</li>
                <li>Unique functionality for managers to create personalized profiles for team members.</li>
                <li>Intuitive, user-friendly interface for managing and updating contact information.</li>
                <li>Authentication and authorization mechanisms using Spring Security for secure access.</li>
                <li>Centralized repository for storing and managing all team-related data.</li>
                <li>RESTful APIs to ensure seamless communication between frontend and backend components.</li>
            </ul>
        </div>

        <h2>Technologies Used</h2>
        <div class="technologies">

                <li><strong>API Design:</strong> RESTful API architecture</li>
                <li><strong>Frontend:</strong> Can integrate with any frontend technology</li>
            </ul>
        </div>
